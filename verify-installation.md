# 安装验证指南

## 🔍 安装后验证步骤

### 1. 基础功能验证

#### 检查扩展是否正常加载
1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 确认"Google Authenticator Extension"显示在列表中
4. 确认扩展状态为"已启用"
5. 确认扩展图标出现在浏览器工具栏

#### 检查弹出窗口
1. 点击扩展图标
2. 确认弹出窗口正常打开
3. 确认显示"Google Authenticator"标题
4. 确认显示"+ 添加账户"按钮
5. 确认显示"已保存账户"区域

### 2. 添加账户功能验证

#### 手动添加账户测试
1. 点击"+ 添加账户"按钮
2. 输入测试密钥：`JBSWY3DPEHPK3PXP`
3. 输入账户名称：`测试账户`
4. 点击"添加账户"
5. 确认账户出现在列表中
6. 确认显示6位验证码
7. 确认倒计时器正常工作

#### 验证码功能测试
1. 等待验证码更新（最多30秒）
2. 点击验证码
3. 确认复制成功提示
4. 粘贴到文本编辑器验证内容正确

### 3. 高级功能验证

#### 筛选功能测试
1. 在搜索框输入账户名称
2. 确认筛选结果正确
3. 清空搜索框
4. 确认显示所有账户

#### 删除功能测试
1. 点击账户的"⋯"菜单
2. 点击"删除账户"
3. 确认账户从列表中消失

### 4. 权限验证

#### 检查权限设置
1. 访问 `chrome://extensions/`
2. 找到"Google Authenticator Extension"
3. 点击"详细信息"
4. 确认以下权限已授予：
   - 存储
   - 活动标签页
   - 脚本
   - 摄像头（可选）

### 5. 错误排查

#### 常见问题检查
- [ ] 扩展图标是否显示
- [ ] 弹出窗口是否正常打开
- [ ] 验证码是否正常生成
- [ ] 倒计时器是否正常工作
- [ ] 复制功能是否正常
- [ ] 筛选功能是否正常
- [ ] 删除功能是否正常

#### 如果遇到问题
1. 检查Chrome版本是否支持（需要88+）
2. 确认开发者模式已启用
3. 尝试重新加载扩展
4. 检查浏览器控制台是否有错误信息
5. 参考[常见问题](INSTALL.md#常见问题)部分
