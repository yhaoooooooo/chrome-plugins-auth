# 常见问题解答

## 🔧 安装问题

### Q1: 扩展无法加载怎么办？
**A:** 请检查以下几点：
- 确保所有必需文件都存在（manifest.json, popup.html, popup.js等）
- 检查manifest.json文件格式是否正确
- 确保Chrome版本支持Manifest V3（Chrome 88+）
- 尝试重新加载扩展
- 检查控制台是否有错误信息

### Q2: 扩展图标不显示？
**A:** 解决方案：
- 检查扩展是否已启用
- 尝试固定扩展到工具栏（右键点击扩展图标）
- 重新加载扩展
- 检查Chrome扩展管理页面中的状态

### Q3: 安装后无法打开弹出窗口？
**A:** 可能的原因：
- 扩展未正确加载
- 权限设置问题
- 浏览器兼容性问题
- 尝试重新安装扩展

## 📱 功能问题

### Q4: 二维码扫描功能不工作？
**A:** 可能的原因和解决方案：
- **权限问题**：确保已授予摄像头权限
- **HTTPS要求**：某些功能需要HTTPS环境
- **浏览器兼容性**：确保使用支持的浏览器版本
- **页面扫描失败**：尝试刷新页面后重新扫描
- **摄像头被占用**：关闭其他使用摄像头的应用

### Q5: 验证码显示错误？
**A:** 检查以下项目：
- 确保系统时间正确（TOTP依赖准确的时间）
- 验证密钥格式是否正确（Base32编码）
- 尝试重新添加账户
- 检查网络时间同步设置

### Q6: 页面二维码扫描找不到二维码？
**A:** 扫描功能支持多种格式：
- **图片元素**：JPG、PNG、GIF、WebP等格式
- **SVG元素**：矢量图形中的二维码
- **Canvas元素**：动态绘制的二维码
- **CSS背景图片**：通过样式设置的背景图片
- 确保二维码清晰可见
- 尝试调整页面缩放比例

### Q7: 导入导出功能不工作？
**A:** 检查以下项目：
- 确保有账户数据可以导出
- 检查浏览器下载权限
- 确认导入文件格式正确
- 查看控制台错误信息

## 🔒 安全问题

### Q8: 数据安全吗？
**A:** 安全特性：
- ✅ 所有数据仅存储在本地浏览器中
- ✅ 不会向任何外部服务器发送数据
- ✅ 使用标准的TOTP算法，确保安全性
- ✅ 权限最小化，仅请求必要的浏览器权限

### Q9: 会收集我的个人信息吗？
**A:** 隐私保护：
- ❌ 不会收集任何个人信息
- ❌ 不会跟踪用户行为
- ❌ 不会上传数据到服务器
- ✅ 所有计算都在本地进行

### Q10: 如何保护我的账户数据？
**A:** 数据保护建议：
- 定期导出账户数据作为备份
- 将备份文件存储在安全的位置
- 不要在不安全的设备上使用
- 定期更新浏览器和扩展

## 🔄 数据管理

### Q11: 如何备份我的账户数据？
**A:** 备份方法：
1. 点击扩展右上角的📤按钮导出JSON文件
2. 使用二维码导出功能生成迁移二维码
3. 将备份文件存储在安全的位置
4. 定期更新备份

### Q12: 如何在不同设备间同步数据？
**A:** 同步方法：
1. 在源设备上导出数据（JSON或二维码）
2. 将备份文件传输到目标设备
3. 在目标设备上导入数据
4. 验证所有账户正常工作

### Q13: 数据丢失了怎么办？
**A:** 恢复方法：
- 如果有备份文件，可以重新导入
- 检查Chrome的本地存储是否被清理
- 尝试重新添加账户
- 联系技术支持获取帮助

## 🎨 界面问题

### Q14: 界面显示异常？

- 使用智能渲染策略减少DOM重建
- 提供更流畅的用户体验

### Q16: 圆形倒计时器不显示？
